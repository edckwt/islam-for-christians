Islam for Christians: Your way to a better understanding of the true message of prophet Jesus.

== Description ==

Islam for Christians is a main informative online source of knowledge about the true message of prophet Jesus and how Muslims view him.

Features:

* Getting the content (articles, videos, audios) of Islamforchristians.com.
* Using Json API
* Using Cron Job

PS: All the websites owned by EDC [(E-Da`wah Committee)](http://www.edc.org.kw), For Support click to [Islam.com.kw](http://www.islam.com.kw/).

== Installation ==

* Upload the Plugin to your blog, Activate it.
* Then the content will be stored automatically in your database.
* The plugin will work automatically using Cron Job to get the recent updates daily or according to the time you set.
